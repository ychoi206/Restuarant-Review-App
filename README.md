##Description
Udacity Front-End Developer Nanodegree Project 7.

##Project Instructions
This restuarant application is converted to a mobile-ready webpage. It is responsive on different sized displays and provides accessibility for those in need. It also has a service worker to provide seamless offline experience for the users.

##Run the Application
In order to run the application:

1. Download the package in git.

2. With your server running, visit the site: http://localhost:8000.
